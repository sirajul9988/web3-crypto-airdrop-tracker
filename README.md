# Web3 Crypto Airdrop Tracker

This is a simple Web3 project.  
It helps you check wallet activity and track possible crypto airdrops.

This project is made for beginners who want to learn how blockchain, wallets, and APIs work.

## What This Project Does
- Connect to a wallet address
- Read blockchain data
- Check recent transactions
- Save wallet history
- Simulate airdrop eligibility check

## Tools Used
- Node.js
- Simple JavaScript
- Public blockchain API
- Local JSON file for data storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- wallet.js = wallet functions
- airdrop.js = airdrop checker
- api.js = blockchain API calls
- data/ = saved wallet data
- logs/ = log files

## Example Use
- Enter wallet address
- App checks transactions
- App shows if wallet is active
- App saves result to file
